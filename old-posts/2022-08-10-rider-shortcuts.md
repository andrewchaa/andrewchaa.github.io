I used to have InteliJ mapping. But I use VS Code so often that I went for VS Code map in Rider.

Btw, these are the symbols for keys on Mac

- Command (or Cmd) ⌘

- Shift ⇧

- Option (or Alt) ⌥

- Control (or Ctrl) ⌃

- Caps Lock ⇪

- Fn

Tests

- Run a test: ⌘ Enter

- Run all tests: ⇧ ⌘ Enter
